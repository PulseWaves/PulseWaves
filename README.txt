The latest PulseWaves version is 0.3 (revision 11) and a DLL for writing and reading is available (including sample programs). The PulseTools folder also contains the first six PulseTools pulseinfo.exe, pulseview.exe, pulse2pulse.exe, pulsezip.exe, pulsesort.exe, and pulseextract.exe can parse various full waveform LiDAR formats (e.g. LVIS, LAS 1.3 FWF, GeoLas Waveform, Optech CSD).
